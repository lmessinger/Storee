How to use Maxxsol_ExtendAPI module in magento
-----------------------------------------------------------------------------

1) Copy the folder 'Maxxsol' to /<magento-root>/app/code/local/
2) Copy the file 'Maxxsol_ExtendAPI.xml' to /<magento-root>/app/etc/modules/

and now you'll be able to see the extended webservices in the magento admin panel.
